# MARKDOWN SYNTAX

## HEADERS
You can use # to make \<h1\> tag to \<h6\> tag
```
This is an <h1> tag
====================
This is an <h2> tag
--------------------

# This is an <h1> tag
## This is an <h2> tag
###, ####, #####, ###### (h3, h4, h5, h6)
```

# This is an \<h1\> tag
## This is an \<h2\> tag
### This is an \<h3\> tag
####### This is what you should not use

## LISTS
### Unordered

Use * , +, - to make unordered list
```
 * Item 1		+ Item 1		- Item 1
 * Item 2		+ Item 2		- Item 2
	* Item 2a		+ Item 2a		- Item 2a
	* Item 2b		+ Item 2b		- Item 2b
```
 * Item 1		
 * Item 2
	 * Item 2a
	 * Item 2b

### Ordered
use number and dot to make ordered list
```
 1. Item 1
 2. Item 2
 3. Item 3
```
1. Item 1	
2. Item 2
3. Item 3
